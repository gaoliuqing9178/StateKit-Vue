## 6. example 已完成第一轮视觉重做

- `examples/vite-vue-admin/src/App.vue`
- `examples/vite-vue-admin/src/styles.css`

当前 example 的方向是：

- 用 6 个 category-first 入口直接展示真实场景，而不是继续堆场景名组件。
- 页面骨架更多依赖留白、分隔线和标题节奏，而不是一层层外部卡片容器。
- CTA、loading、disabled 和 link 行为仍然保持现有 API，不额外引入新的 props 设计。

## 7. docs 示例页和 docs 站外层样式已同步去卡片化

这一轮没有只改 example，也把 docs 站的外层展示一起收口了：

- `apps/docs/src/demo-styles.css`
- `apps/docs/src/styles.css`

当前已经同步到的范围包括：

- 三个 docs example 页共用的 demo 外壳
- docs 首页的 hero 支撑区域
- recipes 列表页的条目容器
- recipe 详情页的 preview / guide / props / actions 区块
- install 页的步骤和侧边信息区
- 顶部导航与 metadata pill 的视觉弱化

整体方向是：

- 减少玻璃卡片和大圆角包裹
- 保留必要强调，但更多依赖分隔线、排版和节奏来组织信息
- 让组件本体成为视觉中心，而不是让外层容器抢戏
